## Description
#game-statistics-api - Dockerized

## Installation

1. Clone this repo
2. Make sure you have docker installed and running on your computer
3. Run `docker-compose up` (you may have to run `docker compose up --build` for the first setup phase)

To access backend's bash:
Run `docker-compose exec app bash`

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
